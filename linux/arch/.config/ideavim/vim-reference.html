⚡ Register Hygiene — The Core Concept

The Problem
yiw → vap → p
p pastes the deleted text, not your yank. Delete overwrote the unnamed register.
Black Hole Register
"_d "_c "_x
Write-only void. Anything sent here never overwrites your yanks.
Paste Over Selection Fix
vmap p "_dP
"_d = delete selection to void. P = paste from still-intact unnamed register.
System Clipboard
"+y "+p
+ register = system clipboard. Keep internal yank separate from OS clipboard.
